# 投資組合管理工具

個人投資組合管理工具，同時管理台股（TWD）與美股（USD）。  
Supabase 雲端儲存，GitHub Pages 托管，登入即可跨裝置使用。

---

## 存取方式

- **URL**：[https://calvin99tw.github.io/portfolio-manager/](https://calvin99tw.github.io/portfolio-manager/)
- **登入**：輸入 Email → 收取 Magic Link → 點擊連結 → 自動登入（session 預設一週）

## 核心設計理念

### 雙資金池模型

工具以「**台股池（TWD）**」和「**美股池（USD）**」兩個獨立資金池來管理投資。
這個模型對應真實世界 — 資金從可動用資金（銀行）轉入券商成為「投資池」，
標的買入從池扣款，賣出與股息回流到對應池。

```
[可動用資金 TWD] ←─ (工具不管理，只是來源概念)
       ↓ ↑
[台股池 TWD]      [美股池 USD]    (兩池獨立，各自原幣管理)
       ↓ ↑                ↓ ↑
   [台股持倉]         [美股持倉]
```

**工具只管兩個投資池的活動，不管理可動用資金、池間轉移、新資金移入紀錄。**

### 原幣別呈現

持倉表格、批次明細、賣出紀錄、股息紀錄全部以**原幣別**顯示。台股用 TWD，美股用 USD，不做即時換算。  
只有「合併檢視」模式才會用設定匯率將美股換算為 TWD 加總。

### 主題與顏色語意

**Dark / Light / System 三段切換**：標題列右側可切換，預設跟隨系統設定。

**顏色語意（亞洲市場慣例）：**
- 🔴 **紅色** = 正向 / 上漲 / 獲利 / 過熱
- 🟢 **綠色** = 負向 / 下跌 / 虧損 / 超賣
- 🟣 **紫色** = 強調 / 選取 / 主要操作

## 功能清單

### 持倉管理
- 新增買入（支援同標的分批）
- 同名標的合併顯示，可展開查看每批明細
- 編輯、刪除批次或整個標的
- 一鍵更新某標的的現價（同步所有批次）
- 以原幣別管理（TWD / USD），不混合計算

### 股價與匯率自動更新
- **台股／美股股價**：自動抓取 Yahoo Finance（盤中延遲約 15–20 分，收盤價準確）
- **USD/TWD 匯率**：自動抓取中央銀行（CBC）
- App 載入與切回頁面時自動觸發，可點 Header 的 ↻ 手動更新
- 若自動抓取失敗，現價欄位顯示 ⚠，點擊可手動輸入

### 操作邏輯（先選取，再操作）
- 點列選取標的或批次（再點取消）
- 工具列按鈕根據選取狀態自動 enable/disable：
  - 未選取：只有「＋新增買入」可用
  - 選取標的：「更新現價」「加碼」「配息」「刪除整個標的」
  - 選取批次：「更新現價」「編輯」「賣出」「刪除批次」

### 資金池管理
- **入金 / 出金**記錄資金進出，累計計算投入本金
- 自動計算可用餘額 = 投入本金 + 已實現損益 + 已收股息 - 已投入成本
- 總資產 = 持股市值 + 可用餘額
- Pool Card 顯示：總資產、配置 Bar（市值比例）、資金現況、累計報酬
- 「分池檢視 / 合併檢視」切換
- 標的的「佔池比」基於投入本金計算

### 股息記錄
- 點選標的後，從工具列「配息」按鈕記錄
- 紀錄欄位：每股股息、股數、稅前總額（自動）、實領金額、除息日、備註
- 獨立「股息歷史」頁籤，按時間倒序顯示
- 已收股息會計入可用餘額

### 賣出與已實現損益
- 對每批持倉操作賣出（部分或全部）
- 賣出後該批移入「交易歷史」頁籤
- 賣出損益不含股息（兩者獨立記錄）
- 賣出回收金額（已實現損益）計入可用餘額

### 資金明細
- 獨立「資金明細」頁籤，列出所有入金 / 出金紀錄
- 摘要顯示台股池 / 美股池各自的投入本金（入金與出金分開加總）
- 支援編輯與刪除每筆紀錄

### 看板（決策層交接）
- 獨立「📋 看板」頁籤，讀取 Claude 日更/建檔產出的 dashboard.json（設定 → 看板資料 匯入）
- 每檔顯示：結論／為什麼／我在等、各關鍵價（進場/加碼/鎖利/停損/檢視）距現價 %
- 貼線（±3% 內）琥珀高亮；收盤跨線標記（盤中越線顯示「待收盤確認」）；催化劑到期日高亮
- 量能門檻自動判定（台股張/美股股）；持有標的在上、觀察名單在下
- 資料存 Supabase（跨裝置同步），價格由 Yahoo 即時計算，不受 JSON 快照過期影響

### 推播通知（看板掃描）
- 每日收盤後自動掃描（台股 14:30、美股 05:00 台北時間），Cloudflare Worker 發送 Web Push
- 催化劑到期日推播已啟用；貼線/跨線價格推播於驗證期後開啟
- 設定 → 推播通知 啟用（每台裝置各一次）；iPhone 需 iOS 16.4+ 且「加入主畫面」後從圖示開啟
- 一次掃描合併為一則通知（最多 10 行），文案標注所依據的看板產出日期

### 績效歷史
- **📈 績效 Tab**：折線圖顯示總市值走勢 vs 投入成本基準線（TWD 計價）
- 每次更新行情後自動記錄當日快照（同一天最後一筆覆蓋）
- 時間篩選：1M / 3M / 6M / YTD / 1Y / ALL
- 區間報酬率、區間損益金額、最新市值摘要

### 警示
- 過熱閾值（預設 ≥+20%）：建議減碼
- 超賣閾值（預設 ≤-15%）：建議加碼
- 閾值可在設定頁調整

## 計算公式

### 持倉成本與損益（原幣）
```
總成本     = 股數 × 買入均價 + 買入手續費
市值       = 股數 × 現價
未實現損益 = 市值 - 總成本
報酬率 %   = 未實現損益 / 總成本 × 100
```

### 賣出損益（含手續費和交易稅）
```
實領金額   = 股數 × 賣出價 - 手續費 - 交易稅
買入成本   = 股數 × 買入均價 + 該批次手續費（按股數比例攤分）
已實現損益 = 實領金額 - 買入成本
報酬率 %   = 已實現損益 / 買入成本 × 100
```

### 同標的合併
```
均攤成本 = Σ(各批次股數 × 各批次買入價) / 總股數
加權天數 = Σ(各批次股數 × 各批次持有天數) / 總股數
```

### 年化報酬
```
持倉中：年化 = ((市值 + 已收股息) / 總成本) ^ (365 / 加權天數) - 1
已賣出：年化 = (賣出總額 / 買入總成本) ^ (365 / 持有天數) - 1
       （賣出年化不含股息，股息獨立累計）
※ 持有天數最少以 1 天計算
```

**短期持有警示**：持有天數 < 30 天時，年化報酬以淡灰色 + ⚠ 顯示，提示數字為數學推算，參考價值有限。

### 資金池
```
投入本金       = Σ 入金 - Σ 出金（依幣別分別計算）
可用餘額       = 投入本金 + 已實現損益 + 已收股息 - 已投入成本
總資產         = 持股市值 + 可用餘額
標的佔池比     = 該標的成本 / 投入本金 × 100
```

### 累計報酬
```
累計報酬       = 未實現損益 + 已實現損益 + 已收股息
累計報酬率     = 累計報酬 / 投入本金 × 100
```

### 合併檢視（換算 TWD）
```
合併投入本金 = TWD投入本金 + USD投入本金 × 匯率
合併總資產   = TWD總資產 + USD總資產 × 匯率
（其他項目同理）
```

## 設定項目

| 設定 | 說明 |
|------|------|
| 入金 / 出金 | 在持倉頁工具列記錄資金進出，投入本金由累計紀錄推算 |
| USD/TWD 匯率 | 自動抓取（CBC 中央銀行）；可手動覆蓋；僅用於合併檢視換算 |
| 過熱警示閾值 | 預設 +20%，超過顯示過熱警示 |
| 超賣警示閾值 | 預設 -15%，低於顯示超賣警示 |

## 資料儲存機制

- **主資料**：儲存於 Supabase（PostgreSQL 雲端資料庫），登入後即時同步，跨裝置共享
- **快取層**：localStorage 作為快取，加速頁面載入、支援短暫離線操作
- **離線時**：頂部顯示警告橫幅，復線後自動同步至 Supabase
- **本機備份**：Settings → 本地備份 可匯出 CSV，可用 Excel 開啟或存檔備份

---

## 已知限制

- **盤中股價不自動輪詢**：股價於 App 載入 / 切回頁面時自動抓取，或手動點 ↻；不會每隔幾分鐘自動更新
- **池間轉移與池總額變動歷史**：暫不記錄
- **績效歷史從開始使用起累積**：無法回補過去資料
